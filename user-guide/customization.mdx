## Changing the text color

Changing colors is a one-click operation:

<Steps>
  <Step title="Open the color menu">
    Click the quote bubble icon → **"Text Color"**
  </Step>
  
  <Step title="Select your preferred color">
    Choose from:
    - **🟡 Yellow** - Bright and visible
    - **⚪ White** - Clean and modern
    - **🔴 Red** - Urgent and attention-grabbing
    - **🟢 Green** - Calming and positive
  </Step>
  
  <Step title="Color is applied immediately">
    The overlay will update in real-time to show your selected color.
  </Step>
</Steps>

<Note>
Your color preference is saved to `~/.menubartext_color` and persists across app restarts.
</Note>

## How color settings are stored

The color selection is saved using a simple file-based system:

```swift
func saveColor(_ color: String) {
    let colorFilePath = NSHomeDirectory() + "/.menubartext_color"
    try? color.write(toFile: colorFilePath, atomically: true, encoding: .utf8)
}
```

From `menubar_text_controller.swift:277-280`

The color value is written as plain text: `"yellow"`, `"white"`, `"red"`, or `"green"`.

## Text customization

Beyond colors, you can customize the actual text content:

### Text length and format

- **Length**: Supports any length from short phrases to longer text
- **Characters**: Full Unicode support including emojis 🎉
- **Special characters**: All special characters are supported
- **Formatting**: Plain text (no bold, italic, or other formatting)

## Display settings

While you can't customize these directly, here's what MenuBarText uses for text display:

| Setting | Value | Purpose |
|---------|-------|--------|
| **Font** | System font | Ensures consistency with macOS |
| **Size** | 14pt | Optimized for menu bar visibility |
| **Alignment** | Left-aligned | Natural reading flow |
| **Spacing** | Proper padding | Prevents text from touching screen edges |

These settings are optimized for readability and consistency with the macOS interface.

## Manual color file editing

<Warning>
Advanced users only: You can manually edit the color file, but using the menu bar controls is recommended.
</Warning>

If needed, you can manually set the color by editing the file:

```bash
# Set color to red
echo "red" > ~/.menubartext_color

# Valid values: yellow, white, red, green
```

After editing, restart the overlay for changes to take effect:

1. Stop Text (⌘X)
2. Start Text (⌘S)

## Combining customizations

For the best experience, consider matching your color and text:

- **Green + Wellness text**: "Remember to take a break 🌿"
- **White + Professional text**: "Quarterly review preparation"
- **Yellow + Motivation text**: "You're doing amazing! ⭐"
